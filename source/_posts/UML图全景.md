## 🍳 做一顿饭 - UML图全景对比

| UML图类型 | 关注点 | 做饭的例子 |
|-----------|--------|-----------|
| **用例图 (Use Case)** | 谁要做什么 | 👤顾客："我要吃晚饭"<br>👤厨师："做晚饭" |
| **活动图 (Activity)** | 怎么做（流程步骤） | 买菜→洗菜→切菜→炒菜→装盘 |
| **状态图 (State)** | 东西的状态变化 | 菜的状态：生的→洗净→切好→炒熟→装盘 |
| **类图 (Class)** | 有什么东西和关系 | 📦厨师（属性：等级、经验）<br>📦菜品（属性：名称、价格）<br>📦厨具（属性：类型） |
| **交互图 (Interaction)** | 谁和谁配合 | 厨师→告诉助手→洗菜<br>助手→回复→洗好了<br>厨师→开始炒菜 |
| **对象图 (Object)** | 具体实例 | 👨‍🍳张师傅（厨师的一个实例）<br>🥬白菜（菜品的一个实例） |

---

## 详细举例：

### 🔷 **类图 (Diagrama de Clase)**
**关注：系统中有哪些"东西"及其关系**

```
┌─────────────┐
│   厨师      │
├─────────────┤
│ - 姓名      │
│ - 等级      │
│ - 工龄      │
├─────────────┤
│ + 做菜()    │
│ + 切菜()    │
└─────────────┘
       │
       │ 使用
       ↓
┌─────────────┐
│   厨具      │
├─────────────┤
│ - 类型      │
│ - 品牌      │
├─────────────┤
│ + 加热()    │
└─────────────┘
       │
       │ 继承
       ↓
┌─────────────┐
│    炒锅     │
└─────────────┘
```

---

### 🔶 **状态图 (Diagrama de Estado)**
**关注：一个对象的状态怎么变化**

**例子：一盘菜的生命周期**

```
[开始]
   ↓
(生的食材) ─[洗净]→ (洗净的食材)
   ↓                      ↓
              [切好]→ (切好的食材)
                          ↓
              [炒制]→ (正在炒)
                          ↓
              [完成]→ (熟菜)
                          ↓
              [装盘]→ (成品菜)
                          ↓
                      [结束]
```

**触发条件：**
- 洗净：水龙头打开
- 炒制：锅温度达到180°C
- 完成：炒制时间>5分钟

---

### 🔷 **活动图 vs 状态图的区别**

#### **活动图：关注"做什么动作"**
```
[开始]
   ↓
[买菜] → [洗菜] → [切菜] → [炒菜] → [装盘]
                                         ↓
                                     [结束]
```

#### **状态图：关注"菜的状态"**
```
[生的] → [洗净] → [切好] → [炒熟] → [装盘]
```

**关键区别：**
- 活动图：动词（买、洗、切）
- 状态图：形容词（生的、熟的）

---

### 🔶 **对象图 (Diagrama de Objeto)**
**关注：某个时刻的具体实例**

```
某天下午5点的厨房快照：

张师傅: 厨师
├─ 姓名 = "张三"
├─ 等级 = "高级"
└─ 工龄 = 10年

宫保鸡丁: 菜品
├─ 名称 = "宫保鸡丁"
├─ 价格 = 38元
└─ 状态 = "正在炒制"

1号炒锅: 炒锅
├─ 品牌 = "苏泊尔"
└─ 温度 = 200°C
```

---

### 🔷 **交互图 (顺序图 Sequence)**
**关注：谁先谁后、怎么配合**

```
顾客      服务员      厨师      助手
 │          │          │         │
 │─点菜────>│          │         │
 │          │─下单────>│         │
 │          │          │─洗菜───>│
 │          │          │<─洗好───│
 │          │          │─炒菜    │
 │          │<─菜好了──│         │
 │<─上菜────│          │         │
```

## 📊 总结对比表

| 图类型 | 看的是 | 做饭比喻 |
|--------|--------|----------|
| **类图** | 静态结构（蓝图） | 厨房设备清单 |
| **对象图** | 某时刻快照 | 今天5点厨房里有什么 |
| **状态图** | 一个东西的变化 | 菜从生到熟的过程 |
| **活动图** | 做事的步骤 | 做菜的工序 |
| **交互图** | 多个对象配合 | 厨师和助手怎么分工 |

---

**记忆口诀：**
- **类图**：有什么（名词）
- **状态图**：怎么变（一个对象）
- **活动图**：怎么做（流程）
- **交互图**：怎么配合（多个对象）
